1. Propositional Logic Basics

What is a Proposition?

A proposition is a statement that is either true or false, but not both.

‚úÖ Valid Propositions:

  • "Python is a programming language" ‚Üí TRUE
  • "5 + 3 = 10" ‚Üí FALSE
  • "Every even number greater than 2 can be expressed as the sum of two primes" ‚Üí TRUE/FALSE (Goldbach's conjecture)

‚ùå NOT Propositions:

  • "What time is it?" (question)
  • "Write some code!" (command)
  • "x + 5 = 10" (depends on x value)

üíª Programming Connection

# In programming, propositions become boolean expressions
x = 5
is_positive = x > 0  # True
is_even = x % 2 == 0  # False

# These are the building blocks of conditional logic
if is_positive and not is_even:
    print("x is a positive odd number")

Direct Proof

Start with assumptions, use logical steps to reach the conclusion.

Theorem: If n is even, then n¬≤ is even

1

Given: n is even

2

4

Conclusion: n¬≤ = 2m where m = 2k¬≤, so n¬≤ is even ‚àé

def verify_theorem(n):
    """Verify: if n is even, then n¬≤ is even"""

Mathematical Induction

Prove for base case, then prove that if true for n, it's true for n+1.

Theorem: Sum of first n integers = n(n+1)/2

Base

n=1: 1 = 1(1+1)/2 = 1 ‚úì

IH

Inductive Hypothesis: Assume true for n=k
1 + 2 + ... + k = k(k+1)/2

Step

Prove for n=k+1:
1 + 2 + ... + k + (k+1) = k(k+1)/2 + (k+1)
= (k(k+1) + 2(k+1))/2
= (k+1)(k+2)/2 ‚úì

üíª Recursive Implementation

def sum_first_n(n):
    """Sum of first n integers using formula"""
    return n * (n + 1) // 2

def sum_first_n_recursive(n):
    """Recursive version mirrors induction"""
    if n == 1:  # Base case
        return 1
    return n + sum_first_n_recursive(n - 1)  # Inductive step

# Verify
for i in range(1, 11):
    formula = sum_first_n(i)
    recursive = sum_first_n_recursive(i)
    assert formula == recursive
    print(f"n={i}: sum={formula} ‚úì")
